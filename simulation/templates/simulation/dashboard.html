Investigating LTE-U & WiFi Coexistence

This framework explores how Multi-Agent Deep Reinforcement Learning (MADRL) can enhance spectrum efficiency and ensure fairness when LTE-U and WiFi networks operate in the same unlicensed frequency bands.

📡

The Coexistence Challenge

LTE-U uses a scheduled, persistent channel access method, which can dominate the unlicensed spectrum. WiFi relies on a "listen-before-talk" (CSMA/CA) mechanism. Without a coordination strategy, LTE-U transmissions can block WiFi indefinitely, a problem known as "WiFi starvation," leading to poor performance and unfair resource allocation.

🧠

The MADRL Solution

MADRL provides an intelligent solution. Multiple agents (e.g., LTE-U base stations) independently learn optimal behaviors by observing the network environment and receiving rewards or penalties. They learn to adapt their transmission patterns (like duty cycles) to maximize overall system performance while ensuring WiFi networks get a fair chance to communicate, all without explicit coordination.
